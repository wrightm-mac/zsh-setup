# Setup for a **ZSH** Terminal Environment

Everything needed to setup & use my zsh environment.

## Prerequisites

Firstly, install the following:

* **zsh**
* **git**
* **wget**
* **tree**
* **figlet**
* **fortune**
* **cowsay**

Change the shell for your user (don't need to sudo for this):
    which zsh
    chsh

## Copy the RC File

    cp ./files/.zshrc ~
